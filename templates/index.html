// pipeline

tokenization.py

Tokenize & build vocab

Lower-cased word tokenizer with a Counter-backed vocabulary. Special tokens <pad>, <unk>, <bos>, <eos> reserve the first four indices.

data_loader.py

Batch & pad

Context windows are flattened for EmbeddingBag; parallel sentences are padded to equal length with pad_sequence for the translation task.

embeddings.py

CBOW & Skip-gram

Two Word2Vec architectures trained side by side on the same toy corpus — compare how each shapes the resulting vector space.

seq2seq_model.py

Encoder – Decoder

An LSTM encoder compresses a source sentence into a hidden state; the decoder unrolls it back out token by token, with teacher forcing during training.

// architecture notes

CBOW context → EmbeddingBag(mean) → ReLU → Linear → target logits
Skip-gram target → Embedding → ReLU → Linear → context logits
Encoder Embedding → LSTM(n_layers) → (hidden, cell)
Decoder <bos> → Embedding → LSTM step → Linear → softmax over vocab → repeat
Evaluation perplexity = exp(cross-entropy) · BLEU-4 with brevity penalty
